# Profile picture

Drop your photo here and name it **`profile.jpg`** (this exact name).

- Recommended: square image, at least 400×400px, JPG or PNG.
- If you use PNG, rename it to `profile.jpg` OR update the `src` in `index.html`
  (search for `assets/profile.jpg`).
- Until you add it, the hero shows a stylish "YP" monogram fallback automatically.
